# 隐私政策

**生效日期**：请根据实际上线日期填写。

## 1. 我们收集的信息

- **账号与资料**：您注册时提供的用户名、密码（加密存储）、邮箱、手机号、昵称等。
- **使用数据**：为提供服务所必需的数据，例如对话内容（AgentChat）、设备绑定信息（若使用个人中心模块）。
- **技术数据**：IP、访问时间、设备类型等，用于安全与运维。

## 2. 信息用途

- 提供注册、登录、聊天、个人中心等功能。
- 保障服务安全、防范滥用与欺诈。
- 在法律要求范围内配合监管。

## 3. 共享与披露

我们不会出售您的个人信息。仅在以下情况可能共享或披露：

- 经您同意；
- 为履行法律义务或响应有权机关合法要求；
- 与为本产品提供基础设施的服务商（如云服务商），且其受合同约束保护数据。

## 4. 安全与保留

我们采用行业通用措施保护数据安全（如加密传输、密码哈希、访问控制）。数据保留期限遵循业务需要与法律要求。

## 5. 您的权利

在适用法律允许的范围内，您有权查询、更正、删除个人数据或撤回同意。具体方式可通过产品内设置或联系运营方。

## 6. 更新与联系

我们可能适时更新本政策并公示。重大变更将通过产品内通知或另行说明。  
如有疑问，请联系运营方（请在此填写联系邮箱或客服方式）。
